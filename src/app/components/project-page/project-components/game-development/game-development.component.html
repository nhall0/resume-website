This is my latest game jam - I learned a lot from my first go in the Summer Jam. By far the trickiest part of any of these is figuring out what to do, then sticking with it. Implementation is the easiest part, followed by asset creation. Originally the game was going to about destroying mechs and recycling them for parts, then creating contraptions to defend against the next wave with their parts. I ended up just making a tower defense game with a mech theme. I'm happy with the result, but I think I could have done more.
